About BlueRound API
===================

The **BlueRound API** provides a powerful interface for managing users, handling authentications, and automating documentation workflows.

API Versioning
--------------

.. note::
   Current stable version: ``v1.2.0``

We use semantic versioning. Major changes that break backward compatibility will be introduced in a new API prefix (e.g., ``/v2/``).

Base URLs
---------

All API requests should be made over **HTTPS**. Calls made over plain HTTP will fail.

* **Production:** ``https://api.blueround.com/v1``
* **Staging:** ``https://api.blueround.dev/v1``
